# ScrapingExample

Simple web scraping program to fetch product name and prices from Seeed Studio Arduino category

## Dependencies

  - [HTTPoison](https://hex.pm/packages/httpoison)
  - [Floki](https://hex.pm/packages/floki)

## Running

```
  mix scrape
```
